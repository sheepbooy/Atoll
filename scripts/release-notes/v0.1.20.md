### 修复
- **macOS**：修复 Claude Desktop 会话跳转按钮错误显示 "Terminal" 的问题
  - 新版 Claude Desktop 与 CLI 共用 `~/.claude/projects/` transcript 路径，不再仅凭路径判定为 CLI
  - Stop / PostToolUse / SubagentStop 等 hook 事件也会检测 session host（此前仅在 PreToolUse 时检测）
  - 所有 hook 事件均通过 peer 进程树区分 CLI 与 Desktop，即使两者同时运行也能正确识别
  - 扩展 Desktop transcript 路径匹配（Application Support、agent-sessions 等）
